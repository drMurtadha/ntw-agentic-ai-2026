# NTW 2026 — Agentic AI untuk Produktiviti Harian

Laman statik bahan latihan untuk GitHub Pages.

## Uji secara tempatan

```bash
python3 -m http.server 8000
```

Buka `http://localhost:8000`.

## Terbitkan melalui GitHub Pages

1. Cipta repositori GitHub baharu.
2. Tolak kandungan folder ini ke cabang `main`.
3. Di GitHub, buka **Settings → Pages**.
4. Di bawah **Build and deployment**, pilih **Deploy from a branch**.
5. Pilih cabang `main`, folder `/ (root)`, kemudian **Save**.

Laman tidak memerlukan proses build atau dependency JavaScript.
