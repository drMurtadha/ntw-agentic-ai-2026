# Demo Workspace — Operasi Pembasmian Tempat Pembiakan Aedes

Folder ini mengandungi data rekaan untuk demonstrasi Agentic AI. Semua nama, tarikh, penugasan dan butiran operasi hanyalah contoh latihan—bukan rekod atau arahan operasi sebenar UTM.

## Struktur

- `demo-files/` — tiga fail input: nota mesyuarat, kehadiran dan maklum balas.
- `demo-output/` — lokasi untuk Codex atau Claude Cowork menghasilkan fail baharu.

## Cara menggunakan dengan Claude Cowork

1. Ekstrak fail ZIP.
2. Dalam Claude Desktop, pilih **Cowork**.
3. Cipta projek dan pilih **Use an existing folder**.
4. Pilih folder `ntw-agentic-ai-demo-workspace` ini.
5. Salin Project instructions dan Task daripada laman latihan.

Jangan gunakan data sebenar, peribadi atau sulit semasa demonstrasi.
